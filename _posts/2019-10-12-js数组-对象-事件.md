# js数组

遍历数组
1.用for循环遍历
```
var arr = [1,2,3,4,5,6,7];
for(var i = 0 i < arr.length; i++){
		console.log(arr[i] + '-----' + i);
}
```
2.用forEach 循化遍历
```
forEach(function (value,index,arrSelf){
		value 当前元素的值
		index 当前元素的索引
		arrSelf 当前数组本身
})
arr.forEach(function(value,index,arrSelf){
		console.log(value + '-----' + index);
})
```
3.用for in 遍历 
```
关联数组 
var obj = {
	uname: 张三;
	age: 30;
	sex: 男;
}
for (var i in obj){
	console.log(i + '----' + obj[i]);
}
for (var i in arr){
console.log(i + '----' + arr[i]);
}
```
